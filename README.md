# Luy

## Project Detail:

**Framework:** NuxtJS (Nuxt 3)  
**Module** Donation Module (An Alumni Hub App Website)

### Key Modules

- **Donation Hub:** A centralized dashboard for impact tracking and campaign discovery.
- **Campaign Wizard:** A 4-step verified process for creating fundraising appeals.
- **Pledge & Automate:** Recurring donation system integrated with GCash and bank cards.
- **Academic Passport:** A verified profile synced with registrar records for secure credential sharing.
- **Biometric Login:** Secure, frictionless access for alumni via FaceID/Fingerprint.
